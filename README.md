# Waston.github.io
## 零代码AI网站编程实战

### 个人情况简介
本人为硬件工程师，具备基础的 C 语言和 HTML 编程能力，仅编写过不超过 500 行的简单驱动代码，无大规模编程及网站开发经验。

### 项目简介
1. **核心开发模式**：本项目全程依托 AI 协助完成（基于 COZE 平台的 AI 网站编程能力），本人不手动编写代码，仅通过向 AI 提问获取建议和可直接复制的代码，再通过「CTRL+C / CTRL+V」的方式在 GitHub 上创建对应文件并粘贴代码，实现零代码开发。
2. **记录核心内容**：全程记录开发全流程，包括遇到的问题、解决思路、技术知识点、项目进展，重点聚焦「AI 提示词设计」「AI 给出的解决方案」「本人的方案选择」，形成可检索、可迭代的实战笔记，为有类似基础的开发者提供参考。
3. **项目状态与目标**：项目尚未完成，本文档将随开发进度持续更新。核心目标是通过该实战巩固编程基础，探索 AI 工具在编程中的应用方法，提升开发效率，掌握小型系统从需求梳理到功能实现的全流程。
